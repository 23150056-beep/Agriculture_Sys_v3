# V4 Glossary

- **Admin**: full authority role with system-wide override.
- **Manager**: planner, approver, and distribution operations owner.
- **Distributor**: farmer/operator field user handling requests and confirmations.
- **Request**: unified order/distribution entity replacing split buyer/dispatch semantics.
- **Approval Queue**: manager-facing list of submitted requests awaiting decision.
- **Delivery Proof**: evidence artifacts confirming handoff/completion.
- **Timeline**: ordered event history of request status and operations.
- **Dual-Run**: temporary period where v3 and v4 endpoints run in parallel.
- **Cutover**: production switch where v4 becomes default and v3 writes are frozen.
